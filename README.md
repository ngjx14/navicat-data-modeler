## Navicat Data Modeler 能做什么？

简单说，这是一款专门用于数据库建模的独立工具，不是 Navicat Premium 里附带的那个，功能更专注、更完整。

**核心能力大概这几块：**

**可视化对象设计器**：表、视图、函数、存储过程，全部用图形化界面操作，不用手写复杂 DDL。支持 Crow's Foot、IDEF1x、UML 三种行业标准符号，随你习惯选。

**三层模型体系**：从概念模型（业务层面的实体关系）→ 逻辑模型（字段、主外键定义）→ 物理模型（对应真实数据库表结构），三者可以互相转换，不用重新画。

**逆向工程**：已有数据库？直接导入，自动生成 ER 图，一下子就能看清楚现有库的结构关系，接手老项目特别省事。

**模型与数据库比对同步**：设计稿改了，数据库还没跟上？用比对功能找出差异，自动生成同步脚本，省去手动对比的麻烦。

**SQL 脚本生成**：按需输出，支持完整性规则、注释、字符集等细节配置，直接拿去执行，不用二次处理。

**Navicat Cloud 支持**：模型文件同步到云端，团队成员共享访问，远程协作不再靠传文件。

支持数据库范围包括 MySQL、MariaDB、PostgreSQL、Oracle、SQL Server、SQLite、MongoDB、Snowflake，基本把主流的都覆盖了。平台方面 Windows、macOS、Linux 全都有。
